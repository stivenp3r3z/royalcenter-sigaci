# Sistema Integrado de Gestión Académica y Calidad de la Información (SIGACI)

## Proyecto de Tesis – Instituto Royal Center Solutions

SIGACI es un sistema web orientado a mejorar la calidad, integridad y confiabilidad de la información académica mediante la automatización de los procesos de:

- Registro de estudiantes
- Gestión de cursos
- Control de asistencia
- Manejo de calificaciones

---

## 🎯 Objetivo General

Desarrollar e implementar un sistema web centralizado que optimice la gestión académica y garantice la calidad de la información en el Instituto Royal Center Solutions.

---

## 🛠 Tecnologías

- Laravel (Framework PHP)
- MySQL (Base de Datos)
- Bootstrap (Frontend)
- GitHub (Control de versiones)
- Metodología Scrum

---

## 📊 Módulos del Sistema

- Gestión de Usuarios
- Registro de Estudiantes
- Gestión de Cursos
- Control de Asistencia
- Gestión de Calificaciones
